Creating Branch in Git

$ git branch  			------------->check the num of branch present

$ git branch <branch_name> 	------------->create a new branch

$ git checkout <branch_name>	------------->switch to new branch

develop your work in this new independent branch

$ git add <filename>

$ git commit -m "message"

$ git remote add origin https://github.com/CHAMAN26/yocto_recipe.git

$ git push origin <branch_name>
